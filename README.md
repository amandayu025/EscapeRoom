# Escape Room
## Description
It's a simple escape room web "game" <br>
It was made with HTML, CSS and JS <br>
This is a project made so i could learn some JS, so i took more time than it should and it's sadly poor, plus, it was the first time i committed through git in vscode :) <br>

<img width="800" height="500" alt="first screen" src="images/first_screen.png">


It's basically the wall the player is seeing, two "arrows" (one for each direction, left and right), the inventory bar and a note space, so the player can organize their clues and conclusions, or whatever they want to remember later. 

## Use instructions (guide to escape the room)
Each wall has a clue, each one guiding to a number, in the end, the player will need to combine them, creating the code to escape the room according to a "guide" (paper explaining the order of the digits), that they can find in the room.<br>
(to make this guide, i chose to move with the right arrow)
